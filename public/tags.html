Manage tags & synonyms

Group synonymous tags so they connect the same entries. Once linked, searching any tag in a group finds entries tagged with any of its synonyms, and the related-entry suggestions treat them as one. Entries keep the exact tag they were given.

Link tags as synonyms

Make the first tag a synonym of the second. If the second tag already belongs to a group, the first joins that group.

Each group shares one canonical tag (shown first). Detach a synonym to make it stand alone again.
